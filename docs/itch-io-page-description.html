Farming Simulator 25 · live data

⬡ Farm Management Dashboard

Your field, your fleet, your money — in one browser, updating while you play. A lightweight in-game mod plus a Windows desktop app that serves the same glassy home experience you get at http://localhost:8766.

Livestock · Vehicles · Fields · Economy · Pastures · Productions — with optional FTP for dedicated / hosted servers.

Stop juggling spreadsheets. Open the dashboard on a second monitor or tablet and run your operation with the same look and feel as the in-app UI.
  • ▸ Mod exports live farm data to data.json from your save or server
  • ▸ App merges Lua export with savegame context when XML is available — real dashboard, not a static page
  • ▸ Built for Windows; local disk paths or FTP for GPortal-style hosts

Quick start

  1. Install the mod, enable it, and load your save once.
  2. Install the Farm Dashboard .exe and run it.
  3. Open the dashboard in your browser; adjust paths or FTP in Settings if needed.
