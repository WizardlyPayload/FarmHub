SimHub read-only page

SimHub’s in-game web overlay cannot click the dashboard. Use this page to show only the block you need (same data as the main app). It tracks the server and farm you last picked in Farm Dashboard on this PC.

Leave empty to list every field card. Use the cluster id shown on each merged field card.

Open on this PC or a tablet: http://<this-PC>:8766/simhub.html (or /simhub). SimHub follows the server and farm you select in Farm Dashboard—no query string needed.

Remote / LAN access

Turn on LAN access below — until then, phones and tablets cannot use http://<this-PC-LAN-IP>:8766 (the server listens on this PC only). After enabling, open that URL from your tablet on the same Wi‑Fi. This app still uses 127.0.0.1 locally without a password. You may need an inbound Windows Firewall rule for TCP 8766. Changing these options restarts the built-in web server.

Other devices time out? If this PC can open its own LAN IP but phones or tablets never load the page, that is almost always the Wi‑Fi/router: use the same network (not guest/IoT), turn off AP / client isolation if your router has it, and check both devices use the same subnet (e.g. 192.168.1.x). Some VPN or security suites block device-to-device traffic even when Windows Firewall is off.

Leave empty to allow any LAN IP (still requires username and password). If set, only listed IPs may connect remotely.
