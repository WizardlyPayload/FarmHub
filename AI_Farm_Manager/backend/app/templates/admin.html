How data reaches this server

Recommended (VPS): turn Push mode On in the form below (writes DASHBOARD_PUSH_MODE=1), leave Farm Dashboard JSON URL empty, and on the PC enable Send farm data to the AI server (Farm Dashboard → Settings → AI Farm Manager → Hosted AI) with the same link key as FARMDASH_INTEGRATION_KEY (or a per-client key from Client connections).

Alternative: G-Portal FTP, or an HTTP URL to Farm Dashboard — must be reachable from this backend (not 127.0.0.1 from inside Docker unless you use host networking).

When On, this server accepts POST /api/integration/push-snapshot from Farm Dashboard. Saves to {{ env_file_path }} and applies immediately (no restart). If your host only injects env at container start and does not mount a writable .env, set DASHBOARD_PUSH_MODE=1 there instead.

Pull path for dashboard JSON when not using push/FTP. For Docker on another machine, 127.0.0.1 points at the container, not your PC — use LAN IP, tunnel, or push mode.
