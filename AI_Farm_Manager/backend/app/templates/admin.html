How data reaches this server

Recommended (VPS): DASHBOARD_PUSH_MODE=1 in this server’s environment, leave Farm Dashboard JSON URL empty below, and on the PC enable Send farm data to the AI server (Settings → AI Farm Manager) with the same link key as FARMDASH_INTEGRATION_KEY above.

Alternative: G-Portal FTP, or an HTTP URL to Farm Dashboard — must be reachable from this backend (not 127.0.0.1 from inside Docker unless you use host networking).
